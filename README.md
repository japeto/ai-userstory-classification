# Feature engineering and ML models for detecting AI-driven functionalities in agile user stories.

This repository contains a Jupyter Notebook implementation for the study “Feature Engineering for Accurate Classification of User Stories with AI-Driven Functionalities.” The project explores how different text representation techniques and feature engineering strategies affect the performance of machine learning models in identifying AI-related functionalities within agile user stories.

## Overview

In agile software development, user stories describe system requirements from the user’s perspective. Automatically identifying AI-driven functionalities within these stories can support:

- Automatic backlog refinement
- Strategic planning of development tasks
- Effort estimation
- Identification of required team skills

This notebook evaluates how lexical, statistical, and contextualized textual representations influence classification performance.
